# Promptless ✦
> Just think. We handle the rest.

Cognitive middleware between your brain and any AI.

## Install
1. Clone: git clone https://github.com/ramjane88/promptless
2. Chrome → chrome://extensions → Developer mode → Load unpacked
3. Add Anthropic API key in popup
4. Go to any AI → type anything → click ✦ Promptless

## Works On
Claude · ChatGPT · Gemini · Grok · Any AI platform
